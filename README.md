# Binary-Classification-using-Machine-Learning
📘 1. Binary Classification using Machine Learning
📌 Project Overview
This project focuses on solving binary classification problems using classical machine learning algorithms. The objective is to compare multiple models and evaluate their performance on real-world datasets such as Titanic Survival and Breast Cancer Diagnosis.
________________________________________
🧠 Models Implemented
•	Logistic Regression
•	Decision Tree
•	K-Nearest Neighbors (KNN)
•	Perceptron
________________________________________
📊 Dataset Description
•	Titanic Dataset – Passenger survival prediction
•	Breast Cancer Wisconsin Dataset – Benign vs Malignant tumor classification
________________________________________
🔧 Methodology
•	Data cleaning and preprocessing
•	Handling missing values
•	Feature scaling and encoding
•	Model training and evaluation
•	Hyperparameter tuning using GridSearchCV
________________________________________
📈 Results
•	Decision Trees performed best for non-linear patterns
•	Logistic Regression showed strong interpretability
•	KNN performed well with optimized k values
________________________________________
🛠️ Technologies Used
•	Python
•	NumPy, Pandas
•	Scikit-learn
•	Matplotlib, Seaborn
